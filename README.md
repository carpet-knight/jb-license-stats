# JB License Stats

Приложение для отображения статистики использования лицензий продуктов JetBrains.

## Backend

#### Установка зависимостей

```shell
$ cd backend && yarn install
```

#### Инициализация БД

```shell
.../backend$ node ./db/init.js
```

#### Запуск

```shell
.../backend$ node ./src/server.js
```

## Frontend

#### Установка зависимостей

```shell
$ yarn install
```

#### Запуск

```shell
$ yarn start
```
